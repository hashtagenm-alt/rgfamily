## 🔬 원인 분석

### 가능한 원인들:

1. **화면이 오래된 캐시 데이터를 표시**
   - 브라우저 캐시나 CDN 캐시가 갱신되지 않음
   - 하드 리프레시(Ctrl+Shift+R) 필요

2. **화면이 다른 데이터 소스를 참조**
   - 시즌 랭킹(season_donation_rankings)을 참조하고 있을 가능성
   - API 엔드포인트 확인 필요

3. **DB 랭킹이 최신 데이터로 갱신되지 않았음** (✅ 해결됨)
   - `refresh-total-rankings.ts` 스크립트 실행으로 해결
   - donations 테이블 데이터를 기반으로 재계산 완료

4. **레거시 데이터 중복 집계**
   - 일부 닉네임이 레거시와 시즌 데이터에 중복 존재
   - 예: "미키™" 레거시 322,030 + 시즌 569,818 = 891,848
